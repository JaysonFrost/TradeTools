# Выключение фонового proxy

## Цель

Дать пользователю явную кнопку выключения локального proxy, когда он запущен, и одновременно отключать настройку его работы после закрытия приложения.

## Поведение

- Кнопка «Выключить proxy» отображается только при активном локальном Xray runtime.
- Нажатие останавливает runtime и сохраняет `keepProxyRunningAfterClose: false`.
- После выключения интерфейс показывает обычное отключённое состояние; «Подключить proxy» остаётся доступной.
- VPN bypass monitor останавливается вместе с runtime по существующему жизненному циклу.

## Границы

- Не меняются серверная цепочка, настройки терминала, VPN-маршруты и другие системные настройки.
- Не добавляются зависимости, фоновые процессы или отдельный переключатель.

## Ошибки

Если runtime не удалось остановить, настройка фоновой работы не изменяется, а ошибка отображается текущим механизмом уведомления интерфейса.
