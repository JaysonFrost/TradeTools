# TradeTools 0.3.9 Release Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Package the accumulated proxy lifecycle changes as the TradeTools 0.3.9 Windows release.

**Architecture:** Keep the existing implementation unchanged. Update only the application metadata and lockfile, then use the project scripts to run the full test/type-check/build pipeline and create the NSIS installer.

**Tech Stack:** Electron, electron-vite, electron-builder, TypeScript, Vitest, npm.

## Global Constraints

- Release version: `0.3.9`.
- Do not add dependencies or change product behavior beyond the accumulated working-tree changes.
- Release artifact: 64-bit Windows NSIS installer.
