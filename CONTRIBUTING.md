# Contributing to TradeTools

Спасибо за интерес к TradeTools. Проект бесплатный, локальный и ориентирован на практичные инструменты для трейдеров.

## Как помочь

1. Создайте issue с описанием проблемы или идеи.
2. Для багов приложите версию TradeTools, ОС, шаги воспроизведения и понятный expected/actual result.
3. Перед pull request запустите проверки:

```bash
cd desktop
npm run typecheck
npm test
npm run build
```

4. Делайте PR небольшими: один баг или одна фича за раз.

## Что не публиковать

Не прикладывайте к issues, PR и скриншотам:

- OBS WebSocket пароль;
- SSH-пароли;
- приватные IP/домены серверов;
- полный `settings.json` без ручной очистки.

## Стиль изменений

- Следуйте существующей структуре `desktop/src/main`, `desktop/src/preload`, `desktop/src/renderer`.
- Секреты должны оставаться в системном keychain, а не в JSON-настройках.
- Новые IPC-методы должны быть узкими и типизированными.
- UI должен оставаться русскоязычным, компактным и удобным для небольших окон.
- Для изменений в видео/прокси-пайплайне добавляйте unit-тесты.
